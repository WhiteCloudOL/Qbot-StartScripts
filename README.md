# Maibot-StartScripts
An easy way to launch MaiBot/AstrBot on Linux&Windows  


```
MaiBot文件分布图
/path/to/your/maibot
├── Maim-with-u/
│   ├── venv/
│   │   └── bin/
│   │       ├── activate
│   │       ├── python
│   │       ├── python3
│   │       └── ...
│   ├── MaiBot/
│   │   ├── bot.py
│   │   └── ...
│   ├── MaiBot-Napcat-Adapter/
│   │   ├── main.py
│   │   └── ...
│   ├── (可选)maimbot_tts_adapter/
│   └── start.sh(下载的启动脚本)
├── 
└── ...
```

```
AstrBot文件分布图
AstrBot
├── venv/
├── data/
├── ...
└── start.sh(下载的启动脚本)
```
